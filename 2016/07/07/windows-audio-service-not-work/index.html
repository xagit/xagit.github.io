音频服务未运行

  • windows
  • 未运行
  • 音频服务
it

现象

如下图，系统托盘处小喇叭上打一个红叉叉，上面提示“音频服务未运行”。

解决办法

重启音频服务即可，重启有2种方法。

命令行操作

Win7以上系统按下Windows+X弹出快捷菜单，然后再按A选择命令提示符（管理员），然后依次输入如下命令即可：

sc stop Audiosrv
s <!-- more --> c start Audiosrv

图形操作

万一我要不记得上面的命令咋办呢？没关系，右键我的电脑，依次选择管理->服务->Windows Audio->重启动此服务，然后就OK了。
